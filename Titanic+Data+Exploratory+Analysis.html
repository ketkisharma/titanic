Inferential Statistics: Since we have seen a strong correlation between gender and survival rate, I am conducting a chi- squared test for the following hypothesis- Null Hypothesis: Gender and Survival are independent. Alternative Hypothesis: Gender and Survival are not independent.

In [14]:
# Chi-Squared Tests
from scipy.stats import chi2_contingency
# Gender to Survivability
pivot = pd.pivot_table(data = titanic_df[['Survived', 'Sex']], index = 'Survived', columns = ['Sex'], aggfunc = len)
pivot
print pivot,"\n"

chi2, p_value, dof, expected = chi2_contingency(pivot)
print "Results of Chi-Squared test on gender to Survival."
print "Does gender have a significant effect on Survival?"
print "Chi-Squared Score = " + str(chi2)
print "Pvalue = " + str(p_value)
print "\n"

As we see, the results are significant for all reasonable alphas. We can reject the Null Hypothesis. So gender has a significant affect on the survival rates of the passengers with females having higher survival rates.
